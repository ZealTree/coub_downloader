# Coub Video Downloader

Этот репозиторий содержит три Python-скрипта для скачивания видео с сайта Coub, их обработки и объединения с аудио. Скрипты позволяют загружать видео, опционально зацикливать их под длительность аудио и сохранять результат в указанной директории.

## Требования

- **Python 3.6+**
- Установленные зависимости:
  - `requests`
  - `ffmpeg-python`
  - `validators` (для первого скрипта)
  - `tqdm` (для второго скрипта)
- Установленный **FFmpeg** (добавьте его в PATH для корректной работы).

Для установки зависимостей выполните:
```bash
pip install requests ffmpeg-python validators tqdm
```

## Установка FFmpeg

1. Скачайте FFmpeg с [официального сайта](https://ffmpeg.org/download.html).
2. Установите его и добавьте в переменную PATH вашей системы:
   - **Windows**: добавьте путь к `ffmpeg/bin` в системные переменные.
   - **Linux/MacOS**: убедитесь, что `ffmpeg` доступен в терминале (проверьте командой `ffmpeg -version`).

## Скрипты

### 1. `coub_downloader_basic.py`
Базовый скрипт для скачивания видео с Coub.

- **Особенности**:
  - Скачивает видео и аудио отдельно.
  - Спрашивает пользователя о необходимости зацикливания видео под длительность аудио.
  - Объединяет видео и аудио в один файл с кодеком `libx264`.
- **Использование**:
  ```bash
  python coub_downloader_basic.py
  ```
  Введите URL видео Coub (например, `https://coub.com/view/xxxxxx`) и следуйте инструкциям в консоли.

### 2. `coub_downloader_advanced.py`
Расширенная версия с поддержкой аргументов командной строки и дополнительных функций.

- **Особенности**:
  - Поддержка аргументов для указания имени файла и директории.
  - Параллельная загрузка видео и аудио с использованием потоков.
  - Логирование процесса.
- **Использование**:
  ```bash
  python coub_downloader_advanced.py https://coub.com/view/xxxxxx -o custom_name.mp4 -d D:/CustomDir
  ```
  - `-o` или `--output`: задайте имя выходного файла.
  - `-d` или `--directory`: укажите директорию для сохранения (по умолчанию `D:/CoubDownloads`).

### 3. `coub_downloader_pro.py`
Наиболее продвинутая версия с поддержкой выбора кодеков и прокси.

- **Особенности**:
  - Выбор видеокодека (например, `libx264`, `libx265`, `libaom-av1` и другие).
  - Поддержка прокси для загрузки.
  - Индикатор прогресса загрузки с помощью `tqdm`.
  - Проверка наличия FFmpeg и поддержки кодеков.
- **Использование**:
  ```bash
  python coub_downloader_pro.py -v libx265 -p http://user:password@host:port
  ```
  - `-v` или `--vcodec`: укажите видеокодек (по умолчанию `libx264`).
  - `-p` или `--proxy`: укажите прокси (опционально).
  После запуска введите URL Coub и следуйте инструкциям.

## Общие шаги использования

1. Убедитесь, что все зависимости установлены.
2. Проверьте наличие FFmpeg в системе.
3. Запустите нужный скрипт с соответствующими аргументами или без них.
4. Введите URL видео Coub в формате `https://coub.com/view/xxxxxx`.
5. Ответьте на вопрос о зацикливании видео (`y` для "да", `n` для "нет").
6. Дождитесь завершения загрузки и обработки.

## Пример

Для скачивания видео с использованием `coub_downloader_pro.py`:
```bash
python coub_downloader_pro.py -v libx264
```
Ввод:
```
Введите URL coub видео (или 'exit' для выхода): https://coub.com/view/12345
Зациклить видео под длину аудио? (y/n): y
```
Результат: видео будет скачано, зациклено под аудио и сохранено как `coub_12345_libx264.mp4` в `D:/CoubDownloads`.

## Устранение неполадок

- **Ошибка "FFmpeg не найден"**: Убедитесь, что FFmpeg установлен и добавлен в PATH.
- **Ошибка загрузки**: Проверьте URL или используйте прокси в `coub_downloader_pro.py`.
- **Кодек не поддерживается**: Убедитесь, что выбранный кодек доступен в вашей версии FFmpeg.
